# Multi-Agent-Stock-Analysis

基于 Kimi 2.6 多 Agent 协作 + Python 计算引擎的 A 股持仓每日分析系统。

---

## 功能特性

- **统一持仓看板**：8 只工业金属板块个股 + 上证指数 + 工业金属板块指数
- **六维度趋势评分**：波浪结构 / 趋势线 / ADX / MACD / MA 斜率 / 价均关系
- **S2_v2 次日预测**：10 因子线性加权 + 波动率驱动动态阈值 + 强制二分类
- **历史验证面板**：预测准确率自动追踪与模型对比
- **每日自动更新**：GitHub Actions 工作日 18:05 自动获取数据、计算、部署

## 快速开始

### 本地开发

```bash
npm install
npm run dev
```

### 本地运行数据流水线

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r scripts/requirements.txt

# 需要 KIMI_API_KEY 环境变量
KIMI_API_KEY=sk-your-key python scripts/run_pipeline.py --stage all
```

---

## GitHub 部署

1. 在 GitHub 创建仓库 `Multi-Agent-Stock-Analysis`
2. push 代码到 `main` 分支
3. 配置 Secret：`KIMI_API_KEY`
4. 启用 GitHub Pages（Source: GitHub Actions）
5. 访问 `https://<你的用户名>.github.io/Multi-Agent-Stock-Analysis/`
